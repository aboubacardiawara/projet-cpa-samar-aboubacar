# CPA
## Pour lancer le projet
### Prérequis 
- nodejs

### Installation
```bash
npm install
```
ou avec yarn
```bash
yarn install
```

### Lancer le projet
```bash
npm run start
```
ou avec yarn
```bash
yarn start
```
